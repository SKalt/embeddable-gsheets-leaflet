Consider whether using google sheets is appropriate

Hosting via google sheets provides a free way to host geodata that changes frequently and provides a user-friendly way to manually update data. If you are interested in hosting geodata that is

  • private
  • more than 500 views per 100 seconds (the api limit)
  • more than a few hundred rows
google sheets may not be a viable solution.

Acquire, Secure a Google Sheets API Key

Go to your google cloud credentials dashboard. Click create credentials > api key.

Confirm key creation by clicking 'restrict key'.

Go to your newly created api key and restrict its use to your intended site.

Note that anyone can use this key on your whitelisted site(s) to call the google sheets api. The key will not give users any permissions. To prevent api quota theft, Google recommends keeping API keys out of front-end.

Configure data handling

A1 notation is in the form 
"sheet id"!A1:B2
, where A1 and B2 are column letter, row number pairs. See the google sheets API for the full A1 notation documentation.
