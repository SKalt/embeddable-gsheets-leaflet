Consider whether using google sheets is appropriate

If you are interested in hosting geodata that is more

  • than 500 views per 100 seconds (the api limit)
  • than a few hundred rows
google sheets may not be a viable solution. If you are looking for an easy way to host geodata that changes frequently and provide a user-friendly way to update data, this might be worth it.

Acquire, Secure a Google Sheets API Key

Go to your google cloud credentials dashboard. Click create credentials > api key. Confirm key creation by clicking 'restrict key'. Go to your newly created api key and restrict its use to your intended site.
